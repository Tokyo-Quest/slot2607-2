# 画像素材マニフェスト(サイズ・ファイル名・格納場所)

- **格納場所(すべてここ)**: `prototype/assets/images/`
- キャラ基準画(生成用・ゲームには使わない)のみ: `prototype/assets/images/_ref/`
- 透過=背景を抜いたPNG / 不透明=PNG or JPG。サイズは書き出し推奨px。
- 「流用」= 別の素材と同一ファイルを使い回せる(生成枚数を減らせる)。

## 枚数の目安
- 背景(不透明): 11枚(残念賞グレー含む)
- 顔(400×400): 4枚(多くの画面で流用)
- 立ち絵(800×1200): 通常4＋キメ4＝8枚(＋一閃の蕎麦戦闘1・アマガミの明媚差分は別サイズ)
- UI/エフェクト/小物: ロゴ・バッジ・PUSH・筐体・銘板・チャンス目3・ノーツ/リング・歯3＋小物・窓・ダイヤル・ゲージ/マーカー・斬撃・敵 ほか

## 参考(音声も同様の置き場)
- サウンドは `prototype/assets/sounds/` に置き、`config.js` の `sounds` でファイル名を指定。
